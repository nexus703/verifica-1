Istruzioni HTML e CSS Utili
HTML

 <head> → Contiene metadati e collegamenti a risorse esterne.
 <meta name="viewport" content="width=device-width, initial-scale=1.0"> → Rende il sito responsive.
 <title> → Definisce il titolo della pagina.
 <link rel="stylesheet" href="stile_gnocchi.css"> → Collega il file CSS esterno.
<style> → Contiene CSS interno.
<body> → Contiene il contenuto visibile della pagina.
 <header> → Intestazione della pagina.
 <div class="container"> → Contenitore principale della pagina.
 <img src="gnocchi.jpg" alt="Gnocchi al pesto"> → Mostra un'immagine con testo alternativo.
 <h1>, <h2>, <h3>, <h4> → Intestazioni di diversi livelli.
 <ol> → Lista ordinata (numerata).
 <ul> → Lista non ordinata (punti).
 <li> → Elemento di una lista.
 <strong> → Testo in grassetto.
 <em> → Testo in corsivo.
 <blockquote> → Citazione evidenziata.
 <cite> → Riferimento a una fonte.
 <q> → Citazione breve.
 <sub> → Testo in apice (es. formule chimiche).
 <sup> → Testo in pedice (es. esponenti).<pre> → Mostra testo preformattato (es. codice).
<table> → Crea una tabella.
 <caption> → Titolo della tabella.
 <th> → Intestazione di colonna.
 <td> → Cella della tabella.
 <tr> → Riga della tabella.

 <u> → Sottolineato.
